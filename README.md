# IT Support Operations Lab

**Service Desk Skills | Ticketing Systems · Active Directory · Network Diagnostics · OS Troubleshooting**

Hands-on lab documentation covering core IT Support and Service Desk operations. Each section contains practical exercises, ticket simulations, and troubleshooting workflows completed in real tools including ServiceNow, Jira Service Management, and Active Directory lab environments.

This repository complements my [Security Analyst Journal](https://github.com/beatrizmartinsc/Security-Analyst-Journal) — together they document my path from IT support foundations to cybersecurity specialisation.

---

## Topics Covered

### 01 — Ticketing Systems
- Incident creation, triage, and resolution in **ServiceNow** (developer instance)
- Service Request and Change Request workflows in **Jira Service Management**
- Professional work notes and resolution documentation
- Knowledge Base article creation

### 02 — Active Directory
- User account lifecycle: creation, password resets, account unlocks, group membership, disabling
- Organisational Units (OUs), Security Groups, and Group Policy concepts
- Identity verification and security-first account management

### 03 — Network Diagnostics
- CLI tools: `ipconfig`, `ping`, `nslookup`, `tracert`, `netstat`
- Structured connectivity troubleshooting: local network → gateway → external IP → DNS
- VPN troubleshooting and escalation documentation

### 04 — OS Troubleshooting
- **Windows:** Task Manager, Event Viewer, Credential Manager, Disk Cleanup, Safe Mode, Print Spooler
- **macOS:** Activity Monitor, Keychain Access, Disk Utility, Force Quit, Safe Mode
- Real-world ticket scenarios with documented resolution steps

### 05 — ITIL & SLA
- ITIL processes: Incident Management, Problem Management, Change Management, Service Request Fulfilment
- Impact × Urgency priority matrix
- SLA targets and ticket queue prioritisation

### 06 — Communication & Escalation
- HEAT method for handling difficult users (Hear, Empathise, Apologise, Take action)
- Functional vs. hierarchical escalation procedures
- Professional documentation standards for ticket handoffs
